# SYS-EMANUEL 🏫

Sistema de gestión escolar desarrollado como proyecto académico de Ingeniería de Sistemas.

## 🚀 Tecnologías Utilizadas

- **Backend:** Java con Spring Boot
- **Base de Datos:** PostgreSQL
- **Contenedores:** Docker
- **Modelado:** PlantUML

## 📊 Casos de Uso Desarrollados

Actualmente el proyecto cuenta con los siguientes módulos de comportamiento:

- **CU01:** Iniciar Sesión
- **CU02:** Registrar Asistencia
- **CU03:** Pago de Mensualidad
- **CU04:** Registrar Notas _(Añadido recientemente)_

---

_Nota: Para visualizar los diagramas de arquitectura y comportamiento, asegúrese de tener instalada la extensión de PlantUML en VS Code._
